# NODEAPP-BOOTCAMP
## Practice WEB-API | Node.js | MongoDB

### To start the application use:

```sh
npm install
```

In production:

```sh
npm start
```

In development:

```sh
npm run dev
```

### Initialising the DB

To initialise the DB to the initial state, you can use the command:

```sh
npm run initDB
```

**WARNING** *This will erase all data from the DB and load the initial state.*
